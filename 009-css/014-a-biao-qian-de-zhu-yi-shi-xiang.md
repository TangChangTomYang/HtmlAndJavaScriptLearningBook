

#### a 标签的注意事项

#### 一 a 标签的假链接
```
<a href="#"> a 标签 # , 回到页面顶部, 页面不刷新</a> <br>
<a href="javascript:">a标签 javascript: , 假链接</a><br>
<a href="">a标签  空连接 ,回到页面顶部 刷新页面</a><br>

<a href="javascript:;">a标签 javascript:; , 禁用a 标签跳转功能</a><br>
<a href="javascript:void(0)">a标签 javascript:void(0) , 禁用a 标签跳转功能</a><br>

```
