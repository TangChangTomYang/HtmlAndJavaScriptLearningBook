#### 内边距



1 、 什么是内边距? 
边框和内容之间的间距就是内边距 (内边距会增加标签占用的区域)

2、Padding 练习的方式
```
padding : 上 有 下 做;
padding : 20px 30px 40px 50px
```
3、padding 分开写
```
padding-top : 10px;
padding-right : 20px;
padding-bottom : 30px;
padding-left : 40px;
```

![](/assets/padding.png)

4、padding 连写 缩写值的说明

padding : 上 右 下 左; 省略 

上 右 下 > 左和右相同
上 右  >  上下相同,左和右相同
上 >  上下左右相同


5、注意: 
(1)、padding 的颜色和标签的背景色是一样的.
(2)、 如果给标签设置了 padding之后,标签所占有的高度和宽度都会有所增加
