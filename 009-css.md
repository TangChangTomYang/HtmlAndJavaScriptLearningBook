##CSS 


**在企业开发中修改样式都是交给CSS来做**

- 通过CSS 来修改样式有什么好处?
(1) 不用记忆那些属性属于哪个标签
(2) 当需求变更时我们不需要修改大量的代码就可以满足需求.
(3) 在前端开发中CSS只有一个作用,就是修改样式.

****
<br>
####一 CSS 的格式

```
<style type="text/css">
    属性名 :属性值;
</style>
```

- 注意点:<br>
(1) style 标签必须写在head 标签的开始标签和结束标签之间.<br>
(2) style 标签中的type属性其实可以不用写 默认就是text/CSS<br>
(3) 设置样式必须按照滚定格`key:value;`式来设置 , 其中`;`不能省略,在最后的可以省略,但是不要省略.



****
